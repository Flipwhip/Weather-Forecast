# Weather Forecast

Погодное веб-приложение на Vue 3 + Vite с прогнозом на 3 дня.

## О проекте

Учебный проект, реализованный в процессе освоения Vue 3 и Vite  
после предыдущего опыта работы с Vue 2 и Vue CLI.  
Цель — практика Composition API, компонентного подхода и работы с внешним REST API.

## Функциональность

- Прогноз погоды на 3 дня с переключением между днями
- Поиск города через поле ввода
- Отображение текущей температуры, влажности, скорости ветра и других параметров
- Иконки погодных условий
- Обработка ошибок (город не найден и др.)
- Данные загружаются при старте для города по умолчанию (Москва)

## Стек

- [Vue 3](https://vuejs.org/) (Composition API, `script setup`)
- [Vite](https://vitejs.dev/)
- [WeatherAPI](https://www.weatherapi.com/) — источник погодных данных
- CSS-переменные, Google Fonts (Montserrat)
- ESLint + eslint-plugin-vue + eslint-config-prettier

## Компоненты

| Компонент    | Описание                                                    |
| ------------ | ----------------------------------------------------------- |
| `App`        | Корневой компонент, загрузка данных, provide города         |
| `PaneLeft`   | Левая панель с фоновым изображением и основными данными дня |
| `PaneRight`  | Правая панель со списком дней и поиском                     |
| `DayCard`    | Карточка одного дня прогноза                                |
| `CitySelect` | Поле поиска города с кнопкой                                |
| `Stat`       | Отображение одного показателя (лейбл + значение)            |
| `Button`     | Переиспользуемая кнопка                                     |
| `Input`      | Переиспользуемый инпут с кастомной директивой `v-focus`     |
| `Error`      | Компонент отображения ошибки                                |

## Запуск локально

```bash
# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev

# Собрать для продакшена
npm run build
```

## API

Проект использует [WeatherAPI](https://www.weatherapi.com/). Для запуска нужен API-ключ.  
Ключ указывается в `src/constans.js` в параметре `key`.

> Бесплатный план поддерживает прогноз до 3 дней — этого достаточно для проекта.
